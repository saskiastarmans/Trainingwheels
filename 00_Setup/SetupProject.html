Setup Visual Studio project / Github

Go to Github and create a empty project (set it up public or private)

You can give your project your own name

  1. Add repository name
  2. Create repository

Go to visual studio

  1. File
  2. New
  3. Project

Search for project

  1. Search for Console app
  2. Select Console App
  3. Give project a name
  4. Give Solution a name (this normally the same name)
  5. Check the checkboxes
  6. OK

Team explorer

  1. Open tab team explorer (right side)
  2. Click on sync
  3. Project

Go to Github

  1. Copy Project name (your own)

Go to Visual studio

  1. Paste in input
  2. Click Publish (you proplably need to add your github credentials)

Go to Github

  1. Hit f5 ==> tada you pushed your project to Github
